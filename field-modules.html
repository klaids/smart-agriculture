    Boards

    We used the STM32 Nucleo-64 boards because are very flexibles and has a good combination of performance and low consumption.
    In particular we used these boards:

  • NUCLEO-L053RB
  • NUCLEO-L152RE
  • NUCLEO-L452RE-P

    As you can see here from this image takes from ST site, our board are developed for works in ultra low power, what it changes is only the flash memory.

    Shield

    For the shield we used P-NUCLEO-LRWAN1 because is compatible with our modules. PREGO AGGIUNGERE ULTERIORI INFO

    IDE

    To run our code we preferred use Arduino IDE, because is very useful, by default this IDE is not compatible with ST Boards, so to manage our boards we used stm32duino, that's a official porting by STMicroelectronics.
    On the Settings Panel of the IDE, you must copy this string:
    https://github.com/stm32duino/BoardManagerFiles/raw/master/package_stmicroelectronics_index.json
    On additional URLs section.
